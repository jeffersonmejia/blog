2. Etiquetas

2.1 Qué son

Una etiqueta define el tipo de información que vamos a estructurar, esta se compone de atributos y propiedades

3. Atributos

Qué son

Son parte de una etiqueta y contienen valores llamados propiedads que modifican el comportamiento de una etiqueta

3. Valores

3.1 Qué son

Contienen valores de un atributo proveniente de una etiqueta del documento HTML
